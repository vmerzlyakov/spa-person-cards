# Скрипты
____

## dev
Сборка и запуск приложения с `hot-reload`, не работает в `IE11` и ниже.
```
npm run dev
```


## dev-ie
Сборка и запуск приложения в любом браузере, но без `hot-reload`.
```
npm run dev-ie
```


## build
Сборка `прод` версии.
```
npm run build
```


## build-dev
Сборка `дев` версии.
```
npm run build-dev
```
